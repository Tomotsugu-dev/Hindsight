【0.5.6】
修复 macOS 多屏幕下焦点采集失准的问题——xcap 的 `is_focused()` 在 macOS 上是"每屏一个 key 窗口"语义，多屏环境下会同时把多个窗口标为焦点，导致日报里"最常用应用"几乎全部错记到 Hindsight 自身。改用 `NSWorkspace.frontmostApplication()` 拿系统层面真正最前的 app PID，再用它筛 xcap 窗口拿标题。Windows / Linux 路径不变。

【0.5.6】
Fixed inaccurate focused-app tracking on macOS with multiple displays — xcap's `is_focused()` follows Cocoa's per-screen "key window" semantics, so on multi-monitor setups several windows came back as focused at once and the iterator deterministically attributed almost all activity to Hindsight itself in the daily report. Now uses `NSWorkspace.frontmostApplication()` to read the canonical OS-level frontmost app PID and filters xcap windows by that PID for the title. Windows / Linux paths unchanged.
