【0.8.8】
- 新增「导出使用数据」（设置 → 数据）：选好日期范围一键导出。Excel 含概览、每日/每周/每月、应用趋势与原始活动明细多张工作表，可筛选、排序、作图；另有 Markdown 与 JSON 格式。
- 「对话」页的索引提示条新增「停止」按钮：手动点「立即回填」建立文字索引后可以随时中断，约 1 秒内停下，已识别的部分保留，之后可随时继续回填。
- 修复「对话」多轮提问从第二轮起回答质量明显下降的问题：此前上一轮的回答会原文带入下一轮，模型容易沿用已失效的引用编号、跳过查询直接改写旧答案。现在每轮提问会先改写成独立完整的问题再重新查证作答，后续轮次与第一轮同样可靠。
- 修复「云同步」页点击"立即同步"后切到其他页面再回来、按钮误显示为空闲的问题：同步进行中的状态现在随时可见，期间重复点击会得到明确提示。

【0.8.8】
- New "Export usage data" (Settings → Data): pick a date range and export in one click. The Excel workbook has overview, daily/weekly/monthly, app-trend and raw activity sheets, ready to filter, sort and chart; Markdown and JSON are also available.
- The indexing banner on the Chat page gains a Stop button: a text-indexing run started manually with "Index now" can now be interrupted at any time — it stops within about a second, everything recognized so far is kept, and indexing can be resumed whenever you like.
- Fixed Chat answers degrading noticeably from the second turn of a conversation: previous answers used to be fed back verbatim, so the model would reuse stale citation numbers or rewrite the old answer instead of querying again. Each follow-up question is now rewritten into a self-contained question first and answered with fresh queries — later turns are as reliable as the first.
- Fixed the Cloud Sync page showing the "Sync now" button as idle after switching pages mid-sync: the in-progress state is now always visible, and clicking again during a sync gets a clear message.

【0.8.8】
- 「使用データのエクスポート」を追加（設定 → データ）：日付範囲を選んでワンクリックで書き出せます。Excel は概要・日/週/月・アプリ推移・生の活動明細の複数シート構成で、フィルター・並べ替え・グラフ化にすぐ使えます。Markdown と JSON も選べます。

- 「チャット」で 2 ターン目から回答品質が明らかに低下する問題を修正：従来は前のターンの回答が原文のまま次のターンに渡り、失効した引用番号を流用したり、再検索せずに古い回答を書き換えたりしていました。現在は各質問をまず自己完結した質問に書き換えてから改めて検索・回答するため、2 ターン目以降も 1 ターン目と同じ信頼性になります。
- 「クラウド同期」ページで「今すぐ同期」中に他のページへ移動して戻ると、ボタンが待機状態と誤表示される問題を修正：同期中の状態が常に見えるようになり、同期中の再クリックには明確なメッセージが表示されます。

【0.8.8】
- Novo "Exportar dados de uso" (Configurações → Dados): escolha um intervalo de datas e exporte em um clique. O Excel traz planilhas de visão geral, diária/semanal/mensal, tendências de apps e atividades brutas, prontas para filtrar, ordenar e criar gráficos; Markdown e JSON também disponíveis.
- A faixa de indexação na página de Chat ganha um botão Parar: uma indexação de texto iniciada manualmente com "Indexar agora" pode ser interrompida a qualquer momento — para em cerca de um segundo, tudo o que já foi reconhecido é mantido e a indexação pode ser retomada quando quiser.
- Corrigida a queda perceptível de qualidade nas respostas do Chat a partir do segundo turno: as respostas anteriores eram reenviadas na íntegra, e o modelo reutilizava números de citação obsoletos ou reescrevia a resposta antiga sem consultar de novo. Agora cada pergunta de acompanhamento é primeiro reescrita como uma pergunta autossuficiente e respondida com consultas novas — os turnos seguintes ficam tão confiáveis quanto o primeiro.
- Corrigida a página de Sincronização mostrar o botão "Sincronizar agora" como ocioso após trocar de página no meio da sincronização: o estado em andamento agora fica sempre visível, e cliques repetidos recebem uma mensagem clara.
