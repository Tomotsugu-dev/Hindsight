【0.8.3】
- 修复开机自动启动可能失效的问题：设置里开着「开机自动启动」时，应用每次启动都会自动校正系统里的自启动注册——0.8.2 清理旧「所有用户」副本时会连带删掉自启动项，本版起自动恢复，无需手动重新设置。

【0.8.3】
- 修復開機自動啟動可能失效的問題：設定裡開著「開機自動啟動」時，應用每次啟動都會自動校正系統裡的自啟動註冊——0.8.2 清理舊「所有使用者」副本時會連帶刪掉自啟動項，本版起自動恢復，無需手動重新設定。

【0.8.3】
- Fixed launch-at-startup silently breaking: with "Start on boot" enabled, the app now re-asserts the OS autostart registration on every launch — the 0.8.2 cleanup of legacy all-users installations also removed the autostart entry as a side effect; from this version it recovers automatically, no manual re-toggle needed.

【0.8.3】
- 起動時の自動開始が無効になることがある問題を修正：「起動時に自動開始」が有効な場合、アプリは毎回起動時に OS の自動開始登録を自動的に修正します——0.8.2 の旧「全ユーザー」コピー削除の副作用で自動開始エントリも削除されていました。本バージョンから自動的に復元され、手動での再設定は不要です。

【0.8.3】
