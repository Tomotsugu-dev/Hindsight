【0.8.7】
- 新增「导出使用数据」：在 设置 → 数据 → 存储 中，可将统计数据导出为文件。日期范围支持最近 7 / 30 / 90 天或自定义起止，内容可勾选每日 / 每周 / 每月统计（各周期含总时长、分类时长与应用时长）。格式三选一：Markdown（推荐，排版好的可读报告，适合直接阅读或存入笔记软件）、CSV（表格长表，Excel / 数据分析工具直接打开做筛选与透视）、JSON（字段最完整，适合程序读取或二次开发）。导出不包含原始活动记录与截图；周 / 月统计与周页、月页显示的数字口径一致。

【0.8.7】
- 新增「匯出使用資料」：在 設定 → 資料 → 儲存 中，可將統計資料匯出為檔案。日期範圍支援最近 7 / 30 / 90 天或自訂起止，內容可勾選每日 / 每週 / 每月統計（各週期含總時長、分類時長與應用程式時長）。格式三選一：Markdown（推薦，排版好的可讀報告，適合直接閱讀或存入筆記軟體）、CSV（表格長表，Excel / 資料分析工具直接開啟做篩選與樞紐分析）、JSON（欄位最完整，適合程式讀取或二次開發）。匯出不包含原始活動記錄與截圖；週 / 月統計與週頁、月頁顯示的數字口徑一致。

【0.8.7】
- New "Export usage data" in Settings → Data → Storage: export your statistics to a file. Pick a date range (last 7 / 30 / 90 days or custom), choose which granularities to include (daily / weekly / monthly — each period covers total time, per-category time and per-app time), and pick one of three formats: Markdown (recommended — a formatted, readable report for direct reading or your notes app), CSV (a flat long table that opens directly in Excel / analysis tools for filtering and pivoting), or JSON (the most complete fields, for programmatic use). Exports contain statistics only — no raw activity records or screenshots; weekly / monthly numbers match what the Week and Month pages show.

【0.8.7】
- 「使用データをエクスポート」を追加：設定 → データ → ストレージ から統計をファイルに書き出せます。期間は過去 7 / 30 / 90 日またはカスタム指定、内容は日次 / 週次 / 月次統計から選択（各期間の合計時間・カテゴリ別時間・アプリ別時間を含む）。形式は 3 つから選べます：Markdown（おすすめ — 整形済みの読みやすいレポート。そのまま読む・ノートアプリに保存する用途向け）、CSV（フラットなロングテーブル。Excel / 分析ツールでそのまま開いてフィルタやピボットに）、JSON（最も完全なフィールド。プログラムでの読み取り向け）。エクスポートに元の活動記録とスクリーンショットは含まれません。週次 / 月次の数値は週・月ページの表示と同じ計算方法です。

【0.8.7】
- Novo "Exportar dados de uso" em Configurações → Dados → Armazenamento: exporte suas estatísticas para um arquivo. Escolha o período (últimos 7 / 30 / 90 dias ou personalizado), as granularidades a incluir (diária / semanal / mensal — cada período cobre tempo total, tempo por categoria e tempo por aplicativo) e um de três formatos: Markdown (recomendado — relatório formatado e legível, para leitura direta ou para o seu app de notas), CSV (tabela longa e plana que abre diretamente no Excel / ferramentas de análise para filtrar e criar tabelas dinâmicas) ou JSON (os campos mais completos, para uso programático). A exportação contém apenas estatísticas — sem registros brutos de atividade nem capturas de tela; os números semanais / mensais correspondem ao que as páginas Semana e Mês exibem.
