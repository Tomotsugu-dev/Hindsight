【0.8.2】
- ⚠️ 重要修复：解决「更新完成后重新打开仍是旧版本」的问题。原因是电脑上可能同时存在「所有用户」与「仅当前用户」两份安装，更新只更新了其中一份。本次起安装时会自动清理旧的「所有用户」副本（需要确认一次系统提权弹窗；数据库与截图都不受影响）。如果本次更新后仍打开到旧版本，请手动卸载旧版一次再安装最新版，此后不会再发生。
- 修复挂机时间被误记为使用时长：显示器息屏、锁屏（Win+L）、屏保启动时立即停止计时，并按最后一次键鼠操作时刻结算——AIDA64 传感器面板这类「画面一直在变」的软件不会再把你离开的几个小时都算成使用。
- 桌面不再延续计时：焦点停在桌面上时按纯键鼠空闲判定，Wallpaper Engine 等动态壁纸不会再让计时永不结束。
- 对话支持「次数」统计：可以问「这周我启动了几次 ××」——相邻使用间隔超过 30 分钟算一次，也可以在问题里自定义（如「离开一小时以上才算一次」）。
- 修复手动导入的模型文件无法被选用的问题：与内置推荐列表同名的模型文件导入后现在直接可选；误导入视觉投影文件（mmproj）时会说明它不能单独作为模型使用。

【0.8.2】
- ⚠️ Important fix: resolved "app still launches the old version after updating". The machine could end up with both an "all users" and a "current user only" installation, and updates only refreshed one of them. Installation now automatically removes the legacy all-users copy (one elevation prompt to confirm; your database and screenshots are untouched). If you still see the old version after this update, uninstall the old copy once manually and install the latest — it won't happen again.
- Fixed idle time being counted as usage: the timer now stops immediately when the display turns off, the session is locked (Win+L), or the screensaver starts, sealing at your last keyboard/mouse activity — software with ever-changing pixels like AIDA64 sensor panels can no longer keep counting for hours after you leave.
- The desktop no longer keeps sessions alive: with focus on the desktop, pure keyboard/mouse idle rules apply, so animated wallpapers (e.g. Wallpaper Engine) can't keep the timer running forever.
- Chat can now count launches: ask "how many times did I launch ×× this week" — a gap of 30+ minutes starts a new session, and you can customize it in the question (e.g. "count it once only if I was away over an hour").
- Fixed manually imported model files not being selectable: files sharing a name with a built-in recommended model are now selectable right after import; importing a vision projector file (mmproj) by mistake now explains it can't be used as a model on its own.

【0.8.2】
- ⚠️ 重要な修正：「更新完了後も再起動すると旧バージョンが開く」問題を解決しました。PC 上に「全ユーザー」と「現在のユーザーのみ」の 2 つのインストールが併存し、更新がその一方だけを更新していたことが原因です。今回からインストール時に古い「全ユーザー」コピーを自動的に削除します（システム昇格の確認が 1 回必要；データベースとスクリーンショットは影響を受けません）。今回の更新後もまだ旧バージョンが開く場合は、一度旧版を手動でアンインストールしてから最新版をインストールしてください。以後は発生しません。
- 離席時間が使用時間として誤記録される問題を修正：ディスプレイのオフ、ロック（Win+L）、スクリーンセーバー起動時に即座に計測を停止し、最後のキーボード・マウス操作時刻で確定します——AIDA64 センサーパネルのような「画面が常に変化する」ソフトが離席中の数時間を使用時間として計上し続けることはなくなりました。
- デスクトップはセッションを延長しなくなりました：フォーカスがデスクトップにある場合は純粋なキーボード・マウスのアイドル判定になり、Wallpaper Engine などの動く壁紙でタイマーが永遠に止まらない問題が解消されます。
- チャットが「回数」の統計に対応：「今週 ×× を何回起動した？」と質問できます——使用間隔が 30 分以上空くと新しい 1 回と数えます。質問内でカスタマイズも可能です（例「1 時間以上離れていたら 1 回と数えて」）。
- 手動で取り込んだモデルファイルが選択できない問題を修正：内蔵推奨リストと同名のモデルファイルは取り込み後すぐ選択可能に；ビジョン投影ファイル（mmproj）を誤って取り込んだ場合は、単独ではモデルとして使用できない旨を説明します。

【0.8.2】
- ⚠️ Correção importante: resolvido o problema de "o aplicativo ainda abre a versão antiga após atualizar". A máquina podia acabar com duas instalações — "todos os usuários" e "somente usuário atual" — e as atualizações renovavam apenas uma delas. A instalação agora remove automaticamente a cópia antiga de todos os usuários (uma confirmação de elevação do sistema; seu banco de dados e capturas não são afetados). Se ainda vir a versão antiga após esta atualização, desinstale a cópia antiga manualmente uma vez e instale a mais recente — não acontecerá de novo.
- Corrigido tempo ausente contado como uso: o cronômetro agora para imediatamente quando o monitor desliga, a sessão é bloqueada (Win+L) ou a proteção de tela inicia, encerrando no momento da última atividade de teclado/mouse — softwares com pixels sempre mudando, como painéis de sensores do AIDA64, não continuam contando horas depois que você sai.
- A área de trabalho não mantém mais sessões vivas: com o foco na área de trabalho, valem as regras puras de ociosidade de teclado/mouse, então papéis de parede animados (ex.: Wallpaper Engine) não mantêm o cronômetro rodando para sempre.
- O Chat agora conta aberturas: pergunte "quantas vezes abri o ×× esta semana" — um intervalo de 30+ minutos inicia uma nova sessão, e você pode personalizar na pergunta (ex.: "conte apenas se fiquei fora por mais de uma hora").
- Corrigidos arquivos de modelo importados manualmente que não podiam ser selecionados: arquivos com o mesmo nome de um modelo recomendado integrado agora ficam selecionáveis logo após a importação; importar por engano um arquivo de projeção visual (mmproj) agora explica que ele não pode ser usado sozinho como modelo.
