【0.7.5】
- macOS 内存占用大幅下降：长时间运行的稳态从约 360 MB 降到约 160 MB，不再持续上涨。
- 「清空所有活动」之后，应用页里那些没图标、没数据的应用行会一起清掉，并同步到其它设备。
- 日 / 周 / 月统计页：点击小时柱（日）或某天（周 / 月）后，排行卡右上角的时间现在显示选中那段子时段的总时长，跟下方应用列表对齐；从占比视图切回时段视图时，之前钉住的大类高亮会自动取消。
- 周 / 月统计页选中某天后的日期标签：中文 / 日文由「6/6」改成「6月6日」；月统计还会加上星期（"6月6日 周六" / "6月6日（土）" / "6/6 Sat"）。
- 窗口高度可拖动调整（640 – 820），宽度保持锁定。
- 日 / 周 / 月统计页底部图例不再显示「隐藏」类别（柱图本来就没这类，图例条目是显示残留）。


【0.7.5】
- Major macOS memory reduction: long-running steady-state drops from ~360 MB to ~160 MB and no longer climbs over time.
- "Clear all activities" now also removes the iconless, empty app rows that used to linger in the Apps page, syncing the cleanup to other devices.
- Today / Week / Month: the duration shown in the top-right of the ranking card now reflects the selected sub-period (an hour for Today, a day for Week / Month) so it lines up with what's listed below; switching from pie back to bars also drops any pinned super-category highlight.
- Week / Month: the date label after clicking a day reads naturally per locale — "6月6日" instead of "6/6" in zh / ja; the Month page also appends the short weekday ("6月6日 周六" / "6月6日（土）" / "6/6 Sat").
- Window can now be dragged taller (640 – 820), width stays locked.
- The bottom legend on Today / Week / Month no longer shows the "Hidden" category (the chart already excluded it; the swatch was a leftover).
