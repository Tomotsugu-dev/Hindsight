【0.6.7】
- 新增「快速模板」总结模式——硬件不够跑本地 vision 模型的用户也能在 AI 总结 → 日报 / 周报 / 月报 看到一份多段叙述风格的回顾，纯 SQL 聚合 + 前端决策树，瞬时生成、无需大模型。
- 总结分 8 段：工作内容概述 / 时间节奏 / 分类构成 / 亮点观察 / 专注度 / 效率评估 / 改进建议 / 小结，结合 top 应用、分类占比、时段分布等数据填空生成。
- 每个分支预置多个变体语句，按日期 hash 稳定挑选——同一天刷新文案不变，不同日感觉更自然。

【0.6.7】
- New "Quick template" summary mode — users without enough hardware for the local vision model can now see a multi-paragraph narrative review under AI Summary → Daily / Weekly / Monthly, generated purely from SQL aggregations and a frontend decision tree. Instant, no LLM required.
- Eight sections per summary: work overview, time pattern, category breakdown, highlights, focus level, efficiency, suggestions, and wrap-up — filled from top apps, category share, and time-of-day distribution.
- Each branch ships multiple preset variants picked by a date hash — phrasing stays stable within a day, but reads more natural across days.
