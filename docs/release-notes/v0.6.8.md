【0.6.8】
- AI 引擎现在可以手动选加速方式——AI 设置 → 引擎页顶部新增「自动 / NVIDIA CUDA / Vulkan / CPU」下拉框；之前装过的 backend 文件保留在磁盘，切回去零等待。
- 新增 Vulkan backend 支持，给 AMD / Intel 显卡 / 没装 CUDA 驱动的 N 卡用户兜底（之前这类用户只能用 CPU 跑总结）。
- 探测不到对应硬件的选项仍可点选（标"未检测到"+ 切换前红色警告），不阻拦明知机器能跑但 Windows 注册表里没标准 ICD 注册的边缘场景。

【0.6.8】
- The AI engine now lets you pick the acceleration backend manually — AI Settings → Engine grows an "Auto / NVIDIA CUDA / Vulkan / CPU" dropdown at the top. Backends you've downloaded before stay on disk, so switching back is instant.
- Added Vulkan backend support as a fallback for AMD / Intel GPUs and NVIDIA cards without a working CUDA driver (previously those users were stuck on CPU summaries).
- Options for hardware Hindsight can't detect are still selectable (with a "not detected" note + red warning on switch), so users on edge cases — like NVIDIA drivers that register Vulkan via non-standard paths — aren't blocked from trying.
