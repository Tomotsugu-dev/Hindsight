【0.7.9】

- New interface themes: Default / Minimal / Dark, with full dark-mode coverage across the app — switch in Settings → Appearance.

- Fixed an issue where captured apps' icons could fail to display.

【0.7.9】
- APIs na nuvem para os resumos de IA: a análise de capturas (visão) e os resumos de texto podem usar provedores separados (ex.: DeepSeek para escrever + Kimi para visão).
- Novos temas de interface: Padrão / Minimalista / Escuro, com cobertura completa do modo escuro em todo o app — troque em Configurações → Aparência.
- Refinamento das estatísticas: a visão de proporções ganha os cartões "média diária" e "comparação com o período anterior".
- Corrigido um problema em que os ícones dos aplicativos capturados podiam não aparecer.
- No macOS, clicar no botão de fechar agora recolhe o app para a barra de menus, em vez de minimizá-lo para o Dock.
