【0.7.1】
- 云同步页可以移除云端「孤儿设备」——其它设备没正常清理就留下的残影。
- 修复：离开电脑锁屏 / 屏保期间，时间不再被错记到「loginwindow」累积成几十分钟"使用时长"。
- 修复：更新说明很长时，对话框会被撑出屏幕导致按钮看不到；现在内容超过屏幕高度时内部滚动，标题和按钮始终可见。
- 修复：云同步过来的远端设备「上次同步」一直显示「—」，现在能正确显示「X 分钟前」等相对时间。
- UI 优化：AI 设置 → 引擎 / 模型 页面重排；今日 / 周 / 月统计切日时柱子不再"漂"动；最常用应用 / 分类列表切日时平滑过渡，不再"跳"到新排名；侧边栏微调宽度避免英文 / 日文标签被截断。
- 「应用分类」页拆成「分类」「应用」两个 tab，分类管理和跨设备应用合并各占一屏，互不打扰。

【0.7.1】
- Cloud Sync page now lets you remove "orphan" devices left behind by other devices that didn't clean up properly.
- Fix: time while the screen is locked or the screensaver is on is no longer mis-attributed to "loginwindow" — those AFK minutes used to pile up as fake "usage time".
- Fix: long release notes used to push the update dialog past the screen edge so the buttons were unreachable; the content now scrolls inside the dialog with the title and buttons pinned.
- Fix: cloud-synced remote devices used to show "—" forever as their "Last synced" time; now displays a proper "X minutes/hours/days ago".
- UI polish: AI Settings → Engine / Models pages redesigned; bars in Today / Weekly / Monthly stop drifting after switching days; top-apps / top-categories lists now slide smoothly into their new ranks instead of snapping; sidebar width nudged so English / Japanese labels stop getting truncated.
- App Categories page split into Categories / Apps tabs — managing categories no longer competes for screen space with multi-device app merging.
