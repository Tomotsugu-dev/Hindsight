【0.7.6-beta】
- 新增巴西葡萄牙语（Português (Brasil)）界面，可在侧栏语言切换里选择。
- 首次安装现在跟随系统语言：之前不论系统语言一律默认中文，现在英文 / 日文 / 葡萄牙语系统打开就是对应语言（其它语言回退英文）；手动选过语言的不受影响，且之后仍会跟随系统语言变化。
- 托盘右键菜单（「显示主窗口」/「退出」）跟随界面语言显示，不再固定中文。


【0.7.6-beta】
- Added a Brazilian Portuguese (Português (Brasil)) interface, selectable from the language switcher in the sidebar.
- First launch now follows your system language: it used to default to Chinese regardless of OS locale; English / Japanese / Portuguese systems now open in the matching language (other locales fall back to English). An explicit language choice is still respected and keeps following the system locale afterwards.
- The tray right-click menu ("Show Window" / "Quit") now follows the app language instead of always being Chinese.
