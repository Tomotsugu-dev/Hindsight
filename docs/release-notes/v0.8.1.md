【0.8.1】
- 修复部分环境下点击「用 Google 登录」浏览器不弹出的问题：登录时若浏览器未自动打开（默认浏览器关联损坏、以管理员运行、被安全软件拦截等），界面会出现「复制登录链接」按钮，粘贴到任意浏览器即可完成登录。
- 修复对话页的误报提示：常驻 OCR 开启时，刚截取、尚在队列中的一两张截图不再被当作"未入索引"提示回填；只有真正滞留超过 10 分钟的积压才会提示。

【0.8.1】
- 修復部分環境下點擊「用 Google 登入」瀏覽器不彈出的問題：登入時若瀏覽器未自動開啟（預設瀏覽器關聯損壞、以系統管理員執行、被安全軟體攔截等），介面會出現「複製登入連結」按鈕，貼到任意瀏覽器即可完成登入。
- 修復對話頁的誤報提示：常駐 OCR 開啟時，剛擷取、尚在佇列中的一兩張截圖不再被當作「未入索引」提示回填；只有真正滯留超過 10 分鐘的積壓才會提示。

【0.8.1】
- Fixed "Sign in with Google" not opening a browser in some environments: if the browser doesn't open automatically (broken default-browser association, running as administrator, blocked by security software, etc.), a "Copy sign-in link" button now appears — paste the link into any browser to complete sign-in.
- Fixed a false alarm on the Chat page: with resident OCR enabled, the one or two just-captured screenshots still in the queue are no longer reported as "not indexed"; the prompt now appears only for a real backlog older than 10 minutes.

【0.8.1】
- 一部の環境で「Google でログイン」を押してもブラウザが開かない問題を修正：ブラウザが自動で開かない場合（既定ブラウザの関連付け破損、管理者権限での実行、セキュリティソフトによるブロックなど）、「ログインリンクをコピー」ボタンが表示され、任意のブラウザに貼り付けてログインを完了できます。

【0.8.1】
- Corrigido "Entrar com o Google" não abrindo o navegador em alguns ambientes: se o navegador não abrir automaticamente (associação de navegador padrão quebrada, execução como administrador, bloqueio por software de segurança etc.), um botão "Copiar link de login" agora aparece — cole o link em qualquer navegador para concluir o login.
- Corrigido um falso alerta na página do Chat: com o OCR residente ativado, as uma ou duas capturas recém-tiradas ainda na fila não são mais reportadas como "fora do índice"; o aviso agora aparece apenas para acúmulos reais com mais de 10 minutos.
