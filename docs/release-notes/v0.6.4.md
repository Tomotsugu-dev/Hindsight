【0.6.4】
Migrated macOS screenshot capture to ScreenCaptureKit, fixing all historical issues on multi-monitor / multi-Space / signed Sequoia release builds (blank captures, desktop-only captures, no captures at all). **macOS 14 (Sonoma) is now the minimum supported version** — earlier macOS releases are no longer supported.
