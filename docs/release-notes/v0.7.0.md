【0.7.0】

- 修复：AI 日报里没截图的时段一片空白，现在会用应用使用记录生成总结。
- 修复：「应用分类」页明明分了类却一直显示「暂无绑定应用」。
- UI 美化：AI 设置 → 模型 / 引擎 页面整体重做；模型卡片加能力标签（TEXT / VISION / FAST 等，hover 有说明）；所有删除 / 下载按钮风格统一。

【0.7.0】
- Fix (important): cross-device sync had a bug in non-UTC timezones (UTC+8, JST, etc.) where the mirror device's session duration stayed at 0 minutes forever. Multi-device users are strongly encouraged to upgrade.
- Fixed: "Clear local database" left the file size untouched at 20+ MB; it actually clears now.
- Fixed: WeChat / QQ / Chrome / Claude / VSCode etc. were showing up in "Top apps" under weird names (like WeChatAppEx) with wrong icons.
- Fixed: AI daily report segments with no screenshots were blank — they now fall back to a summary based on app usage.
- Fixed: "App categories" page kept showing "no apps bound" for categories that already had apps.
- UI polish: redesigned AI Settings → Models / Engine pages; model cards now show capability chips (TEXT / VISION / FAST etc. with hover hints); delete / download buttons unified across the app.
